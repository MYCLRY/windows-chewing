0.3 版

0.2.9 版

  3. 修正了些微的 bug 。

0.2.8 版

  3. 新增選項：設定是否要使用 Esc 鍵，清空編輯區的字串。
  4. 修正打字中切換成英文，會把注音字串清空。
  5. 修正核心，當使用者新增詞之後，不會出現在選字選單的問題。
  6. 狀態視窗新增新按鈕，可直接開啟詞庫編輯器。

0.2.0 版
  1. 修正了右邊數字鍵盤的問題，後來才發現上游 libchewing 本身有提供此功能，但實際使用發現他有問題會造成當機，順便幫上游核心修正了
  2. 修正不少錯誤的詞和注音。 感謝新酷音詞庫團隊
  3. 修正暗黑破壞神下無法正確顯示選字的問題
  4. 修正選字視窗錯誤位置，現在可以支援 Word 直書模式
  5. 修正 Shift 或 Ctrl 在某些程式當中會失效的問題
  6. 修正安裝程式，現在會檢查是否舊版有正確移除，如果沒有會跳警告不給裝
  7. 修正 Freemind 等等 Java 軟體下無法正常使用的問題
  8. seamxr 修正了一些當機問題，以及上游 libchewing 的內部錯誤
  9. 加入可強制關閉狀態工具列的選項 (即使系統設定為顯示狀態列仍然隱藏)
  10. 加入可以預設以全形啟動的選項
  11. 加入可以關閉 Shift 切換中英的選項，讓某些常不小心按到的人可以不要用
  12. 支援新酷音原本就有提供的「前方加詞」，現在可以設定 Ctrl+數字要加入
  13. 游標左邊的詞還是右邊，原本要移動游標到詞前面才能加，改成可加入游標前方的詞，就可以少作幾次移動游標的動作，會比較快
  14. 嘗試支援多螢幕，因為作者群都只有一個螢幕，無法測試是否正常
  15. 支援可修改選字按鍵為 asdf... 而不是預設的 1234，讓有練過指法的人打字更快
  16. 手動加詞時，會顯示提示表示新增的詞，以及成功或失敗
  17. 修改上游 libchewing 和詞庫，現在可以支援打出注音符號 ˇ ˋ ˊ ˙ (本功能目前尚未支援所有鍵盤對應)
  18. 工具列加入按鈕的工具提示
  19. 針對 Windows 98 做出修正，並且修正 libchewing 內部的問題，現在可正常於 Windows 98 使用，已經由 seamxr 修好了
  20. 已經可以在 World of Warcraft 當中打字，但可能有問題，歡迎 WoW 使用者協助測試。
  21. 可變更選字視窗每頁的字數，建議設定為 9 ，因為部分軟體只針對微軟的輸入法設計，把這個數字寫死了，設定其他字數會不能用，例如 Windows xp 的命令列提示字元 cmd.exe
  22. 增加一份使用說明
  23. 改了詞庫格式，並且修正了許多注音錯誤
